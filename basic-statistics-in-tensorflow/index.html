Basic statistics in TensorFlow

08 May 2017

Just a short post with some examples of how to use TensorFlow’s auto-differentiation to compute MLEs.

Gradient descent

Let’s start with a simple example:

  • We want to compute the MLE of a univariate Gaussian, from which we observe an iid sample of size $n = 100$.
  • The true mean and standard deviation are $(10, 5)$.
  • We use gradient descent initialized at $(0, 1)$ with a very fine step size $\alpha = 0.0001$.
import numpy as np
import tensorflow as tf

sess = tf.Session()

TRUE_MU, INIT_MU = 10.0, 0.0
TRUE_SIGMA, INIT_SIGMA = 5.0, 1.0
SAMPLE_SIZE = 100
LEARNING_RATE = 0.0001
NUM_ITERATIONS = 10000

# generate sample
np.random.seed(0)
x_obs = np.random.normal(loc=TRUE_MU, scale=TRUE_SIGMA, size=SAMPLE_SIZE)

mu = tf.Variable(initial_value=INIT_MU)
sigma = tf.Variable(initial_value=INIT_SIGMA)
x = tf.placeholder(dtype=tf.float32)

gaussian_dist = tf.contrib.distributions.Normal(loc=mu, scale=sigma)
log_prob = gaussian_dist.log_prob(value=x)
neg_log_likelihood = -1.0 * tf.reduce_sum(log_prob)

optimizer = tf.train.GradientDescentOptimizer(learning_rate=LEARNING_RATE)
train_op = optimizer.minimize(loss=neg_log_likelihood)

sess.run(fetches=tf.global_variables_initializer())

print(' iter |   mu   | sigma  |   loss')
for i in range(NUM_ITERATIONS):
    sess.run(fetches=train_op, feed_dict={x: x_obs})
    mu_hat, sigma_hat = sess.run(fetches=[mu, sigma])
    obs_loss = sess.run(fetches=neg_log_likelihood,
                        feed_dict={x: x_obs})
    if i % 100 == 0:
        print(' {:4} | {:.4f} | {:.4f} | {:.4f}'.format(i, mu_hat, sigma_hat,
                                                        obs_loss))

print('Fitted MLE: [{:.4f}, {:.4f}]'.format(mu_hat, sigma_hat))
print('Target MLE: [{:.4f}, {:.4f}]'.format(x_obs.mean(), x_obs.std()))
